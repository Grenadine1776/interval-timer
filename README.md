# interval-timer

Simple offline interval timer PWA for iPhone-style home screen use.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current MVP

- English UI
- Work / Rest / Warmup / Cooldown stages
- Reorder stages with `Up` and `Down`
- Remove unused stages with `Remove`
- Per-stage background color
- Configurable round count
- Workout screen with active stage, remaining time, round progress, total delay, pause, next, and exit
- Offline support through PWA caching
